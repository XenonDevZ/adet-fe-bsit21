Introduction

The Academic Consultation Booking System (ACBS) is a web-based platform developed to streamline how students schedule and attend faculty consultations — eliminating the need for informal, uncoordinated arrangements.

What It Is

A school-based web system that manages the full consultation lifecycle — from booking to feedback.

Who It Serves

Designed for students, teachers, and administrators of Liceo de Cagayan University.

Why It Was Built

To replace informal, scattered methods with a centralized, reliable, and real-time solution.

  • Supports both online and face-to-face consultation modes.
  • Built with real-time features — live chat, instant notifications, and file sharing.
Group 2 · ACBS Introduction

Project Overview

What is ACBS?

A web-based booking platform for scheduling academic consultations between students and teachers — online or face-to-face.

Target Users

  • Students — book, track, and attend consultations
  • Teachers — set schedules and approve requests
  • Admins — oversee all users and bookings

Key Features

  • Book face-to-face or online consultations
  • Live chat during active sessions
  • Google Meet link for online consultations
  • Instant notifications for every update
  • File sharing inside the consultation chat
  • Reschedule requests with teacher approval
Group 2 · ACBS Project Overview

Problem Statement

Most schools manage faculty consultations through informal, unstructured methods — causing confusion and missed sessions.

No Centralized System

Students rely on walk-ins, texts, or verbal agreements — leading to scheduling conflicts and no-shows.

Scattered Communication

Conversations spread across multiple apps with no official record per consultation.

Zero Admin Visibility

Administrators cannot track faculty workload or identify which students need more support.

  • Students miss critical academic guidance due to unconfirmed appointments.
  • Teachers are overwhelmed with uncoordinated, informal requests.
